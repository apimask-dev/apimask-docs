# Rate Limits

Rate limits are enforced per API key using a Redis sliding window.

| Tier | Per minute | Per day |
|---|---|---|
| Free | 20 | 500 |
| Starter | 100 | 5,000 |
| Pro | 500 | 50,000 |
| Enterprise | Custom | Custom |

When a limit is exceeded, the API returns `429` with a `RATE_LIMIT_EXCEEDED` error and a
`retry_after` value (in seconds).
