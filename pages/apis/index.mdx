## API categories

ApiMask is a modular API platform. Endpoints are grouped into categories so you can
combine exactly the APIs your product needs. Live categories are available on RapidAPI
today; roadmap categories are in active development.

| Category | Status | What it covers |
|---|---|---|
| Developer utilities | Live | Regex generation and explanation, stacktrace analysis, README and changelog generation, JSON formatting, Base64 encode/decode, hashing, UUID/ULID generation, URL parsing |
| AI content | Live | SEO meta tags, summaries, rewrites, blog outlines, product descriptions, social captions, headlines, FAQs |
| Email & domain | Live | Email validation, bulk list verification, CSV uploads, MX and DNS record lookups, disposable and role-account detection, deliverability checks |
| Website & security | Live | SSL certificate checks, security headers analysis, robots.txt and sitemap validation, CORS checks, URL safety |
| QR codes | Live | URL, WiFi, vCard, email, and SMS QR codes with branded colors and logo overlay |
| SEO | Live | Meta tag generation, site audits, structured-data and metadata checks |
| OCR & PDF | Roadmap | PDF-to-Markdown, structured field and table extraction, OCR cleanup |
| Resume & ATS | Roadmap | ATS scoring, keyword gap analysis, resume rewrite, job matching |
| App Store / ASO | Roadmap | App Store metadata, keyword generation, release notes, review sentiment |
| Data enrichment | Roadmap | Company, email, and domain enrichment |
| AI agents & workflows | Roadmap | Tool endpoints and structured extraction for AI agents |

One ApiMask account and one integration pattern works across every category as new APIs
ship.
