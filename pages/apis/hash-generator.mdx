# Hash Generator API

Generate hashes for a string value. Use it in developer tools, debugging workflows,
integrity checks, and automation systems.

## Endpoint

```http
POST /v1/dev/data/hash
```

## Request body

| Field | Type | Required | Description |
|---|---|---:|---|
| `value` | `string` | Yes | Value to hash. |
| `algorithms` | `string[]` | No | Algorithms to return. Defaults to `md5`, `sha1`, `sha256`, and `sha512`. |

## Example request

```bash
curl --request POST \
  --url https://apimask-developer-utilities-api.p.rapidapi.com/v1/dev/data/hash \
  --header "Content-Type: application/json" \
  --header "X-RapidAPI-Key: $RAPIDAPI_KEY" \
  --header "X-RapidAPI-Host: apimask-developer-utilities-api.p.rapidapi.com" \
  --data '{"value":"hello","algorithms":["md5","sha256","sha512"]}'
```

## Example response

```json
{
  "success": true,
  "data": {
    "hashes": {

  "error": null,

```
