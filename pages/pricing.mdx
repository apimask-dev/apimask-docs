# Pricing

ApiMask pricing is managed on RapidAPI. Each RapidAPI product can have its own plans,
monthly quotas, and overage rules.

[Find ApiMask on RapidAPI](https://rapidapi.com/user/aftaab)

## What RapidAPI handles

- Plan selection
- Payments and invoices
- API keys
- Monthly quotas and rate limits
- Usage dashboards

## What this site provides

This site focuses on product information and detailed API documentation: endpoint
behavior, request schemas, response schemas, examples, and implementation notes.
